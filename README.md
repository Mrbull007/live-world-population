# 🌍 Live World Population Counter (2026)

A beautiful real-time world population dashboard built with pure HTML, Tailwind CSS & Vanilla JavaScript.

![Live Demo](https://mrbull007.github.io/live-world-population/)

### Features
- Real-time population counter (updates every ~50ms)
- Accurate 2026 base value from UN data
- Growth rate: **2.2 people per second**
- Births / Deaths / Net growth today
- Fully responsive & mobile-friendly
- Zero dependencies, no API key required
- Ready for GitHub Pages

### Tech Stack
- HTML5
- Tailwind CSS (via CDN)
- Vanilla JavaScript
- Font Awesome icons

### How to Run Locally
1. Clone the repo
2. Open `index.html` in your browser

### Deployment
- Enable **GitHub Pages** in repository Settings → Pages → Deploy from `main` branch.
